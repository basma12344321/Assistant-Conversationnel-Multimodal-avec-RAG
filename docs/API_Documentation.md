# Documentation API

À compléter au fur et à mesure de l'implémentation (ou générée automatiquement via
`/docs` — Swagger UI fourni par FastAPI sur http://localhost:8000/docs).

## Endpoints prévus

| Méthode | Route | Description | Statut |
|---|---|---|---|
| POST | /api/chat | Envoyer un message, recevoir une réponse RAG + sources | À faire |
| POST | /api/documents/upload | Uploader un document à indexer | À faire |
| GET | /api/documents | Lister les documents et leur statut d'indexation | À faire |
| DELETE | /api/documents/{id} | Supprimer un document | À faire |
| POST | /api/audio/transcribe | Transcrire un fichier audio | À faire |
| POST | /api/images/analyze | Analyser une image | À faire |
| POST | /api/auth/login | Authentification | À faire |
